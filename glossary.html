Glossário

Tags no HTML

elementos HTML h1 - h6
representam seis níveis de título de seção. h1 é o nível de seção mais alto e h6 é o mais baixo.

elemento div
Container genérico para conteúdo de fluxo, que de certa forma não representa nada.

elemento HTML section
Seção genérica contida em um documento HTML, geralmente com um título, quando não existir um elemento semântico mais específico para representá-lo.

elemento HTML hgroup
destina-se a agrupar cabeçalhos de diferentes níveis para uma seção do documento.
